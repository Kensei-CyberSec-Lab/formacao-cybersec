🚨 Lab IDS/IPS - Web Victim

Esta página simula uma aplicação web vulnerável para testar detecções do Suricata.

📋 Endpoints de Teste

GET /

Página principal (tráfego benigno)

curl http://192.168.25.20/

GET /admin

Área administrativa (deve disparar alerta)

curl http://192.168.25.20/admin

POST /login

Formulário de login (teste de regras customizadas)

curl -X POST http://192.168.25.20/login -d "user=admin&password=123456"

GET /search?q=<script>alert('xss')</script>

Parâmetro XSS (deve disparar regra XSS)

curl "http://192.168.25.20/search?q="

🔍 Como Testar

  1. Execute docker exec -it kali_lab bash
  2. Use os comandos curl acima para gerar tráfego
  3. Verifique logs em ./suricata/logs/eve.json
  4. Monitore alertas com docker logs suricata_lab
