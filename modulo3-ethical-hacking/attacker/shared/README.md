# Pasta Compartilhada

Esta pasta é montada em ambos os containers Kali como `/shared`.

Use este diretório para:
- Compartilhar arquivos entre host e containers
- Transferir arquivos entre containers CLI e GUI
- Armazenar flags de CTF e soluções
- Manter anotações e relatórios
- Trocar ferramentas e scripts

## Acesso dos containers:
```bash
cd /shared
```

## Acesso do host:
Este diretório: `./shared/`
